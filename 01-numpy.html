פרק 1 · NumPy ופייתון

קוד פייתון
קריאה והבנה

מבני נתונים בסיסיים, מערכי NumPy, ושאלות קריאת קוד שמופיעות בכל בחינה.

הפרק הזה לא מצריך הרבה תיאוריה — בעיקר תרגול קריאת קוד. במבחן יש כמעט תמיד שאלה אחת על הרצת קוד פייתון: נתון קוד, מה הפלט? צריך לדעת לעקוב אחר list comprehension, set, dict, ולוודא מה כל פעולה עושה.

מבני נתונים שצריך להכיר

📘ארבעת מבני הליבה

1. List — רשימה סדורה, ניתנת לשינוי

סדר חשוב. אפשר לכלול כפילויות. גישה לפי אינדקס.

  • lst.count(x) — סופר כמה פעמים $x$ מופיע.
  • lst.append(x) — מוסיף בסוף.
  • lst[i] — גישה לפי אינדקס.
  • len(lst) — אורך.

2. Set — קבוצה לא סדורה, ללא כפילויות

אין סדר. ערכים ייחודיים בלבד. חיפוש הוא $O(1)$.

  • set([1,1,2,3]) = {1, 2, 3}.
  • s.add(x) — מוסיף.
  • x in s — בדיקה מהירה אם x בקבוצה.

3. Dict — מיפוי מפתח לערך

מפתחות ייחודיים. גישה $O(1)$.

  • d[key] — גישה.
  • d[key] = value — הוספה/עדכון.
  • d.keys(), d.values(), d.items() — איטרציה.

4. Tuple — דומה ל־list, אבל לא ניתן לשינוי

נוצר עם סוגריים: (1, 2, 3). שמיש כמפתחות ב־dict (כי immutable).

נקודה למבחן — אי אפשר להשתמש ב־list כמפתח של dict (כי הרשימה משתנה). חייבים tuple. שורה כמו d[some_list] = 5 תזרוק שגיאה.

List Comprehensions

📘תחביר ופירוש

List comprehension היא דרך מקוצרת ליצור רשימה. התחביר:

comprehension.py
# General pattern:
# [expression for item in iterable if condition]

# Example 1: squares
squares = [x**2 for x in range(5)]
# [0, 1, 4, 9, 16]

# Example 2: with condition
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]

# Example 3: nested
matrix = [[i*j for j in range(3)] for i in range(3)]
# [[0, 0, 0], [0, 1, 2], [0, 2, 4]]
טריק לקריאה: קרא מאחור! קודם for x in range(10), אז if x % 2 == 0, ואחר־כך x. זה מה שיוצר את הפלט.

NumPy — מערכים מהירים

📘למה NumPy? פעולות וקטוריות

NumPy הוא הבסיס לכל ML בפייתון. הוא מספק:

  • מערכים רב־מימדיים (ndarray) — הרבה יותר מהירים מ־lists.
  • פעולות וקטוריות — אופרציה על כל המערך בלי לולאה.
  • פונקציות אגרגציה — np.mean, np.std, np.sum.

פעולות וקטוריות — הקסם של NumPy

numpy_basics.py
import numpy as np

# Create arrays
a = np.array([1, 2, 3, 4, 5])
b = np.arange(0, 10, 2)        # [0, 2, 4, 6, 8]
c = np.zeros(5)                # [0., 0., 0., 0., 0.]
d = np.ones((2, 3))            # 2x3 matrix of ones

# Vectorized ops - no loops needed!
a + 10        # [11, 12, 13, 14, 15]
a * 2         # [2, 4, 6, 8, 10]
a ** 2        # [1, 4, 9, 16, 25]

# Element-wise op between two arrays
e = np.array([10, 20, 30, 40, 50])
a + e         # [11, 22, 33, 44, 55]

# Aggregation functions
a.mean()      # 3.0
a.sum()       # 15
a.std()       # 1.414
a.max()       # 5

# Boolean filtering
a[a > 2]      # [3, 4, 5]

📝תרגיל 1 · practice1, שאלה 1
List comprehension + count

מה הפלט של הקוד הבא?

exercise1.py
from typing import List

def function(param: List[int]):
    return list(set([item for item in param if param.count(item) > 1]))

lst = [1, 2, 1, 2, 3, 4, 5, 1, 1, 2, 5, 6, 7, 8, 9, 9]
print(function(lst))

(ב) ובמשפט אחד — מה הפונקציה עושה?

שלב 1 · ספירה לכל איבר

נחשב param.count(item) לכל איבר:

  • 1 מופיע 4 פעמים
  • 2 מופיע 3 פעמים
  • 3 מופיע פעם אחת
  • 4 מופיע פעם אחת
  • 5 מופיע 2 פעמים
  • 6, 7, 8 מופיעים פעם אחת
  • 9 מופיע 2 פעמים
שלב 2 · סינון (count > 1)

אחרי ה־filter, רק האיברים שמופיעים יותר מפעם אחת נשארים. הרשימה לפני set:

[1, 2, 1, 2, 1, 1, 2, 5, 5, 9, 9]

(שים לב: כל אחד מ־1 ו־2 מופיע מספר פעמים ברשימה הזאת.)

שלב 3 · הסרת כפילויות עם set

set(...) מסיר כפילויות, ואז list(...) מחזיר רשימה.

(א) [1, 2, 5, 9]  ·  (ב) הפונקציה מחזירה את כל הערכים שמופיעים יותר מפעם אחת ברשימת הקלט.
📝תרגיל 2 · practice1, שאלה 2
dict comprehension + zip + reversed

מה הפלט של:

exercise2.py
dictionary = dict()
abc = 'abcdefghijklmnopqrstuvwxyz'
for e1, e2 in zip(abc, reversed(abc)):
    dictionary[e1] = e2

def function(inp1, inp2, d):
    for c1, c2 in zip(inp1, inp2):
        if d[c1] != c2:
            return False
    return True

a = 'data'
b = 'atad'
c = 'wzgz'
print(function(a, b, dictionary))
print(function(b, c, dictionary))
print(function(a, c, dictionary))
שלב 1 · בניית המילון

המילון ממפה כל אות מ־abc לאות במיקום ההפוך:

  • 'a' ← 'z' (כי 'a' ראשון, 'z' אחרון)
  • 'b' ← 'y'
  • 'c' ← 'x'
  • ...
  • 'd' ← 'w'
  • 't' ← 'g'
  • 'g' ← 't'
  • 'w' ← 'd'
  • 'z' ← 'a'
שלב 2 · function(a='data', b='atad', d)

בודק עבור כל זוג: d[a[i]] == b[i]?

  • d['d'] = 'w', אבל b[0] = 'a'. 'w' ≠ 'a' ← False.
שלב 3 · function(b='atad', c='wzgz', d)
  • d['a'] = 'z', c[0] = 'w'. 'z' ≠ 'w' ← False.
שלב 4 · function(a='data', c='wzgz', d)
  • d['d'] = 'w', c[0] = 'w'. ✓
  • d['a'] = 'z', c[1] = 'z'. ✓
  • d['t'] = 'g', c[2] = 'g'. ✓
  • d['a'] = 'z', c[3] = 'z'. ✓
  • כל הבדיקות עברו ← True.
False, False, True.  ·  הפונקציה בודקת האם המחרוזת השנייה היא ה"היפוך" של הראשונה לפי האלפבית (כל אות מוחלפת באות הסימטרית מבחינת המיקום באלפבית).
📝תרגיל 3 · practice1, שאלה 3
מילון עם list כמפתח — שגיאה!

נתון קוד:

exercise3.py
import random

dictionary = dict()
for i in range(10):
    sample = list(random.sample(range(100), i+1))
    dictionary[sample] = len(sample)

מה קורה? האם הקוד רץ בהצלחה? אם לא, מה צריך לשנות?

הקוד לא רץ. השורה dictionary[sample] = len(sample) זורקת:

TypeError: unhashable type: 'list'

הסיבה: פייתון דורש שמפתחות של dict יהיו hashable — כלומר immutable. רשימות (lists) משתנות בזמן ריצה ולכן אי אפשר לחשב עבורן hash יציב.

פתרונות אפשריים:

  • להמיר את ה־list ל־tuple: dictionary[tuple(sample)] = len(sample)
  • להמיר ל־frozenset (אם הסדר לא חשוב): dictionary[frozenset(sample)] = ...
  • להמיר למחרוזת (פחות מקובל): dictionary[str(sample)] = ...
הקוד נכשל עם TypeError: unhashable type: 'list'. הפתרון: להמיר ל־tuple.

סיכום מהיר לזיכרון

📌שש נקודות שאתה חייב לזכור
  1. List — סדור, מאפשר כפילויות. Set — לא סדור, ייחודי.
  2. set(list) — הסרת כפילויות. list(set(x)) חוזר לרשימה.
  3. lst.count(x) סופר בסיבוכיות $O(n)$. אם רוצים מהיר — השתמש ב־Counter מ־collections.
  4. List comprehension — [expr for x in iterable if cond].
  5. NumPy — פעולות וקטוריות בלי לולאות. np.array + 10 פועל על כל איבר.
  6. מפתח dict חייב להיות hashable — list לא יכול להיות מפתח, tuple כן.
